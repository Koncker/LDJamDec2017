This was the game developed for Ludum Dare Game Jam of December 2017

System Requirements:
	Windows 7+ 32/64 Bit

Installation:
	Installation is not required. Unzip the Hangry v1.0 file respective to your PC Specifications (x64 or x86) and run the .exe file
